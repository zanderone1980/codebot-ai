How It Works

Three steps. Zero babysitting. Full audit trail.

1

Point at a GitHub Issue

Run codebot --solve 42 and walk away. CodeBot reads the issue, understands the codebase, and gets to work.

2

Autonomous Fix, Test, Review

CodeBot writes the fix, runs your test suite, and self-reviews its own diff. If tests fail, it iterates. If the review fails, it rejects itself.

3

Get a Production-Ready PR

A clean pull request with passing tests, a detailed description, and a full JSON audit log of every decision made along the way.

Key Features

Enterprise-grade autonomy with safety built into every layer.

Constitutional Safety (CORD)

Every action is risk-scored before execution. High-risk operations are blocked or require approval. Safety is not optional.

Full Audit Trail

Every decision, tool call, and file change is logged to structured JSON. Complete transparency for compliance and debugging.

Runs Locally

Your code never leaves your machine. CodeBot runs entirely on your hardware. Air-gapped environments supported.

Self-Review

After writing a fix, CodeBot reviews its own diff as a separate pass. If the review fails quality checks, it rejects its own work and retries.

32 Built-in Tools

File operations, git, web requests, browser automation, Docker, SSH, test runners, and more. Extensible with custom tools.

Multi-LLM

Works with Claude, GPT-4, or local models via Ollama. Switch providers without changing your workflow.
